# Fedora Web Server

# Purpose

This repository is meant to be an example on how to build a Fedora Web Server using Fedora Bootc and [image-temaple](https://github.com/ublue-os/image-template). It is using  [quadlet](https://github.com/containers/quadlet) to run [Caddy](https://github.com/caddyserver/caddy) as a webserver on [Fedora Bootc 42](https://quay.io/fedora/fedora-bootc:42)
